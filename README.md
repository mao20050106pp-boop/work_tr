→より実務にに多様な環境で学習、経験を行うことが大事だと思った

youtubeでのプロ



AIをメインで使いながら制作してる→AIのうまい使い方も同時に学んでる

youtubeの配信者が行ったlol(ゲーム)の試合を後で遡ることができ、過去にどんなチャンピオンを配信内で使用したことがあるのかと、その配信アーカイブを紐づけ、すぐに統計をみれ、アーカイブを視聴できるwebアプリで開発する。 これを制作物として作りたい
